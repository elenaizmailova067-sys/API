# Курсовая работа «Резервное копирование картинок котиков»

Программа предназначена для автоматического копирования изображений котиков с сайта [CATAAS](https://cataas.com) в облачное хранилище Яндекс.Диск.

## Особенности
- **Загрузка «по воздуху»**: Картинки передаются напрямую из API в облако, не сохраняясь на жестком диске компьютера.
- **Интерактивность**: Пользователь сам выбирает текст, который будет наложен на изображение.
- **Отчетность**: По итогам работы создается JSON-файл с метаданными загруженного файла (имя и размер).
- **Визуализация**: Процесс загрузки отображается с помощью индикатора Progress Bar.

## Требования
Для работы скрипта необходим Python 3.x и библиотеки, указанные в `requirements.txt`.
